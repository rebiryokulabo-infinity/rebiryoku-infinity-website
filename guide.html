テキストエディタについて

HTMLファイルを編集するには「テキストエディタ」が必要です。

おすすめ（無料）

Visual Studio Code（VS Code）
Windowsで使いやすく、日本語対応。カラーで見やすく表示されます。
ダウンロード：code.visualstudio.com

一番簡単な方法

メモ帳でも編集できます
ファイルを右クリック →「プログラムから開く」→「メモ帳」を選択

ニュース記事を追加する

⭐ 一番簡単

js/news-data.js を編集するだけで、記事を追加・削除できます。

1
js/news-data.js をテキストエディタで開く
2
「↓↓↓ 新しい記事はここに追加」と書いてある行のすぐ下に、
以下のブロックをコピーして貼り付ける
3
date・category・title・body を書き換えて保存する

コピーするテンプレート：

{ date: "2026.05.01", ← 日付（年.月.日） category: "お知らせ", ← カテゴリ title: "タイトルをここに書く", ← 記事タイトル image: "", ← 画像なしは "" のまま body: `本文をここに書きます。 段落を分けたい場合は 1行空けてください。` },
カテゴリの選択肢

"お知らせ" ／ "サロンカー" ／ "スクール" ／ "コラム" ／ "キャンペーン"

画像付きの記事にしたい場合

写真を images/ フォルダに入れてから、imageに名前を書きます。

image: "images/my-photo.jpg",
注意

記事と記事の間には必ず カンマ（,）が必要です。
最後の記事だけカンマなしでもOKです。

写真を差し替える

ファイルを置き換えるだけ

方法①：同じファイル名で上書きする（一番簡単）

1
変えたい写真と同じファイル名で、新しい写真を用意する
例：サロンカー.jpg という写真を変えたい場合、新しい写真を「サロンカー.jpg」にリネーム
2
古いファイルに上書きコピーする
移動サロンカーフォルダ内の古いファイルに上書き

方法②：新しい写真を追加して、HTMLのパスを変える

1
新しい写真を website/images/ フォルダに入れる
2
対象のHTMLファイルを開き、Ctrl+F で古い画像名を検索
例：「サロンカー.jpg」で検索
3
src="古いファイル名" を src="images/新しいファイル名" に書き換えて保存

現在の写真の使われ方

../サロンカー.jpg
トップページ ヒーロー背景・サービスカード・アクセスマップ横
../LINE_ALBUM_..._3.jpg
トップページ コンセプト写真
../IMG_2673.jpeg

../LINE_ALBUM_..._5.jpg

基本情報（住所・営業時間・LINE・メール）を変更する

⭐ js/news-data.js だけ編集

js/news-data.js の一番下にある SITE_INFO を編集してください。

// ⚙️ サイト基本情報 — ここを変更してください const SITE_INFO = { name: "Re美力Labo Infinity", ← サロン名 address: "〒674-0065 兵庫県明石市...", ← 住所 hours: "10:00 〜 20:00（不定休）", ← 営業時間 lineId: "@snn9506v", ← LINE ID lineUrl: "https://line.me/...", ← LINE URL email: "メールアドレス", ← メール };
注意

SITE_INFO はフッター・アクセスページ等に自動反映するよう今後追加予定です。
現在はHTMLファイル内も手動で変更してください（Ctrl+H で一括置換が便利です）。

全ページのフッター情報をまとめて変更したい場合

VS Codeで websiteフォルダを開いて、
Ctrl+Shift+H（一括置換）で古いテキストを新しいテキストに一括で置き換えられます。

メニュー・料金を変更する

menu.html を編集
1
menu.html をテキストエディタで開く
2
Ctrl+F で変更したいメニュー名を検索
例：「キャビテーション」で検索
3
以下の部分を探して書き換える
<div class="menu-item"> <div class="menu-item-name">メニュー名</div> <div class="menu-item-desc">説明文</div> <div class="menu-item-price">¥ 12,000<span>（税込）</span></div> </div>

新しいメニューを追加する場合

上のブロックをまるごとコピーして、隣に貼り付けて内容を書き換えてください。

各ページのテキストを変更する

各HTMLファイルを直接編集

HTMLファイルを開くと、テキストが書かれた部分にコメントマークが付いています。

<!-- ✏️ EDIT: キャッチコピー ここから --> <p class="hero-catchcopy"> 内側から輝く、あなた本来の美しさへ。<br> ← ここを編集 サロン・サロンカー・スクールで<br> トータルビューティをご提供します。 </p> <!-- ✏️ EDIT: ここまで -->

このコメントを目印にテキストを探して書き換えてください。

ヒント

<br> は改行です。文中に入れると改行されます。
<strong>太字</strong> で太字になります。
タグ（< >で囲まれた部分）は変更しないでください。

変更をNetlifyに反映する

ファイルを編集したら、Netlifyに再アップロードすることでサイトに反映されます。

1
app.netlify.com にアクセスしてログイン
2
自分のサイトを選んで「Deploys」タブを開く
3
websiteフォルダ全体を画面にドラッグ＆ドロップ
ファイル1つではなく、websiteフォルダごとドロップしてください
4
「Deploy」と表示されたら完了。数十秒でサイトに反映されます
ポイント

写真を変更した場合も、必ずwebsiteフォルダ全体をアップロードしてください。
写真は website/images/ フォルダに入れてからアップロードするのが確実です。
